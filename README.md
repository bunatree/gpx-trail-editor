# GPX Trail Editor

GPXファイルを読み込んで、地図上でポイントやルートを直感的に編集・修正できるWebアプリです。

## ⛰️ こんな「山行記録のトラブル」を解決できます

登山アプリやGPSデバイスの記録でよくあるトラブルを、数クリックで修正できます。

- **「あ、記録し忘れた！」**（スタート地点からしばらくの間、軌跡がない）
  - ➔ 「先頭追加モード」で、実際の開始地点まで軌跡を遡って追加できます。
- **「ログが飛んでしまった…」**（アプリの停止などで、軌跡が直線的に飛んだ）
  - ➔ 「挿入モード」で、正しいルート上にポイントを補完。時刻や標高も自動計算されます。
- **「ゴールまで記録できていない」**（バッテリー切れなどで、途中で記録が途絶えた）
  - ➔ 「末尾追加モード」で、下山地点まで軌跡を延長できます。
- **「ログを止め忘れて、バスに乗っちゃった」**（下山後の移動が余計な軌跡になった）
  - ➔ 「以降削除機能」を使って、不要なログを途中から一気にカットできます。

## 🚀 使い方

1. `index.html` をブラウザで開きます。
2. 画面のドロップ領域に `.gpx` ファイルをドラッグ＆ドロップしてください。
3. 地図上にポイントとルートが描かれ、データテーブルに詳細が表示されます。

### ポイントの移動（移動モード）
地図左側の「ポイント移動」ボタンをクリックすると、ポイント移動モードが ON になり、地図上のポイントをドラッグして移動できるようになります。

### ポイントの追加・挿入
地図左側の「ポイント追加/挿入」ボタンをクリックするたびに、以下のモードが切り替わります。

1. **末尾追加モード**: 地図をクリックすると、ゴール地点の先に新しいポイントを追加し、ルートを延長します。
2. **先頭追加モード**: 地図をクリックすると、スタート地点の前に新しいポイントを追加し、ルートを前方に伸ばします。
3. **OFF**: 通常の状態に戻ります。

また、既存のポイントの間（途中）に挿入したい場合は、地図上のポイントをクリックして表示される吹き出しの **[この後に追加/挿入]** ボタンをクリックしてください。

### 日時と標高の補完
新しく追加・挿入されたポイントの通過時刻は、前後のポイントとの距離や標高差から自動で計算（補間）されます。また、テーブルのメニューから国土地理院の標高APIを使用して正確な標高を取得することも可能です。

## 🛠️ その他の機能
- **ルート反転**: スタートとゴールを入れ替えます。
- **日時シフト**: 記録全体の時間を一括でずらします。
- **軌跡の平滑化**: GPSの誤差によるジグザグを軽減します。
- **ポイントの間引き**: 肥大化したログを指定間隔で軽量化します。
